css

navegar na DOM

  • clique com o direito > inspecionar (abre o elemento na DOM)
  • pesquisar na DOM (CTRL + F), aceita:
    • string
    • seletores CSS ex.: section[class="hiddean"]
    • XPath queries ex.: //section/ul/li
  • Referenciar um elemento pelo js na aba console:
    • inspect(document.querySelector('section'))

Styles tab

Mostra todas as declarações css para o elemento selecionado. Útil para para analisar a ordem de cascata

  • Selecione um elemento para ver seu css
  • Filtro geral (propiedades, valores, classes, id)

computed tab

não precisa ver a ordem de cascata? Olhe o computed

mostra quais propriedades estão aplicadas no elemento selecionado

  • clica na declaração pra ver a cascata própria daquela propriedade
  • filtro geral

prototipando css - classes

  2. no botão :cls

prototipando css - pseudo-classes

como usar:

  1. na aba de styles
  2. no botão :hov

mantém um estado especial:

  • :hover
  • :active
  • :focus-within
  • :focus
  • :visited

Relação de contraste (texto e fundo)

indica se o texto está legível com a cor de fundo atual.

  1. inspeciona um elemento
  2. na aba de styles
  3. na declaração de color
  4. clica no quadrado ao lado do valor da cor
